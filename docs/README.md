# SpendWise - GitHub Pages Documentation

This directory contains the GitHub Pages site for SpendWise.

## Deployment

To deploy this site to GitHub Pages:

1. Push the `docs/` folder to your repository
2. Go to repository Settings â†’ Pages
3. Set source to "Deploy from a branch"
4. Select branch: `main` (or your default branch)
5. Select folder: `/docs`
6. Save

The site will be available at: `https://[username].github.io/[repository-name]/`

## Local Development

Simply open `index.html` in a web browser to view the site locally.

## Content

- **Introduction**: Overview of SpendWise features
- **Architecture**: MVVM architecture and design patterns
- **UI Tour**: Screenshot placeholders and feature descriptions
- **Demo Video**: Placeholder for demo video embed
- **Technology Stack**: List of technologies used

## Customization

Edit `index.html` to:
- Add actual screenshots (replace placeholder divs)
- Add demo video embed code (YouTube, Vimeo, etc.)
- Update repository URL
- Add more sections as needed
